## 2. 프로젝트의 목적과 최종 목표

- **비전**: "한국인을 위한 아시아 최고의 여행 예약 플랫폼" — 글로벌 OTA가 못 채우는 한국인 맞춤 경험(원화 투명가격 + 완전 한국어 + 아시아 특화 큐레이션).
- **핵심 사업모델(2026-05-20 확정)**: **골프텔 중심 모델**. 몽키트래블식 골프+호텔 패키지가 메인 SKU.
  - 메인 SKU = **골프텔**(골프+호텔). 항공 결합형은 별도 상품 언어 **`golftel-air`**(가칭 "골프투어")로 분리 관리.
  - 일반 호텔·항공 단품·항공+호텔(동적 결합, `airhotel.html`)은 보조 SKU.
- **타깃**: 일본·베트남·태국·필리핀 골프 여행을 즐기는 한국인(20~50대).
- **최종 목표(중장기)**: 프론트 프로토타입 → 실 백엔드(ELLIS) 연동 → 실 결제·실 재고·실 채널 매니저 연동으로 실서비스화.

## 3. 현재 상태

- **단계**: **인터랙티브 클릭 프로토타입 완성 단계**. 실제로 클릭하면 동작하는 mock 데모이며, 실 결제·실 재고·실 백엔드 연동은 **전 단계**.
- **형태**: **빌드 없는 순수 정적 사이트**(Vanilla HTML/CSS/JS). npm 빌드·서버·DB **없음**.
- **규모(2026-07-14 실측)**: HTML **40개**, JS **4개**(`data.js`/`state.js`/`icons.js`/`sw.js`), MD 문서 **13개**.
- **git**: `main` 브랜치, 작업 트리 **clean**(미커밋 0), 원격과 동기화됨. 최신 커밋 `fa6d73a`.
- **배포**: GitHub Pages(main 자동). 최신 작업까지 **push 완료**(라이브 반영됨).
- **데이터**: 실 DB/API 없음. 전부 `data.js`(목업) + `state.js`(브라우저 localStorage). **비밀키·`.env` 없음**.
- **백엔드(ELLIS)**: 실제 코드 없이 **MD 명세 문서로만** 존재(의도된 설계).

> ⚠️ **주의**: `PRD.md`·`ARCHITECTURE.md`는 **미래 지향 목표 스택**(Next.js/TS/Supabase)을 적어둔 것으로, **현재 실제 코드와 다릅니다.** 실제는 정적 프로토타입입니다.
